# Sprint Walkthrough Example

End-to-end sprint execution as an illustration of Ch. 03 and Ch. 11.

## Flow
1. Sprint PRD approved (`docs/30-sprint-prds/**`).
2. DoR verified via `../../checklists/definition-of-ready.md`.
3. Implementation with PRs following `../../templates/pull-request.md`.
4. Testing per Ch. 15; observability per Ch. 16.
5. DoD verified via `../../checklists/definition-of-done.md`.
6. Sprint report filed via `../../templates/sprint-report.md`.
