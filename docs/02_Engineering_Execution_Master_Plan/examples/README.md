# Examples

Populated in Phase 4. Each subfolder holds worked, non-normative examples:

- `module/` — worked EEMP application to a real module (MOD-001 Workspace).
- `workflow/` — end-to-end sprint walkthrough.
- `prompt/` — exemplar AI prompts against Chapter 13 standards.
- `review/` — exemplar code / security / performance / a11y reviews.
- `testing/` — exemplar test plans and results.

Examples are illustrative, not authoritative. They must cite the standard they demonstrate.
