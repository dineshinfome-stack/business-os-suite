# Program Status Reports

This directory holds all issued program status reports.

## Naming

- `YYYYMMDD_status_report.md` — weekly
- `WAVE_<X>_EXIT_REPORT.md` — wave-exit
- `YYYYMMDD_adhoc_<slug>.md` — ad-hoc
- `PROGRAM_STATUS_BASELINE_<TS>.md` — baseline snapshots

## Rules

Reports are **immutable once issued**. Corrections are issued as new reports referencing the superseded one.

See `../STATUS_REPORT_TEMPLATE.md` and `../STATUS_REPORT_CADENCE.md`.
