# 17 — Quality Gates

## Entry Criteria (per sprint)
- Sprint PRD approved.
- Dependencies at required maturity.
- Test plan referenced.

## Exit Criteria (per sprint)
- Unit tests ≥ threshold defined in EEMP Ch. 15.
- Integration tests green.
- Documentation updated.
- Security lint clean.
- Performance budget met.

## Wave Gates
Milestone Exit Checklist (`indexes/milestone_exit_checklist.md`) signed before the next wave begins.
