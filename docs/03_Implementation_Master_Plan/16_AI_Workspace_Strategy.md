# 16 — AI Workspace Strategy

## Timing
AI Workspace (MOD-018) rolls out in Wave H, after Analytics (MOD-017) is Beta and Waves A–G core workflows are stable.

## Rollout
1. Knowledge indexing over stable module data (Analytics indexes).
2. Prompt integration governed by EEMP Ch. 13.
3. Agent capability rollout module-by-module.
4. Workflow automation last (requires stable workflow engine).

## Dependencies
- MOD-017 Analytics — for knowledge substrate.
- MOD-001 Workflow Engine — for automation.
- AI Gateway per platform integration.

## Quality Gates
Every AI capability passes the six controls in EEMP Ch. 14 before release.
