# 06 — Milestone Plan

## Phases

| Phase | Waves | Milestone Target | Exit Criteria |
|---|---|---|---|
| P0 Platform Foundation | A | MOD-001 GA | Auth, Org, RBAC, Audit, Settings, Notifications, Search live |
| P1 Master Data | B | MOD-005/006/007 GA | Item master, customer master, employee master usable |
| P2 Commercial | C | MOD-003/004/015 GA | Quote-to-cash and procure-to-pay flows usable |
| P3 Finance | D | MOD-002/008 GA | Ledger, statements, payroll runs live |
| P4 Operations | E | MOD-019/009/010 GA | Warehouse execution, MFG, Projects live |
| P5 Service | F | MOD-011/012/016 GA | AMC, Field Service, Service Desk live |
| P6 Assets & Fleet | G | MOD-013/014 GA | Asset register and fleet ops live |
| P7 Intelligence | H | MOD-017/018 GA | Analytics dashboards and AI workspace live |

## Milestone Exit Checklist (applied to every wave)
See `indexes/milestone_exit_checklist.md`. Summary: Architecture review · API contracts finalized · DB migrations validated · Unit tests passing · Integration tests passing · Documentation updated · EEMP quality gates satisfied · Approval recorded.

## Success Criteria
Each phase closes only when every module in its wave reaches the GA definition in Ch. 05.
