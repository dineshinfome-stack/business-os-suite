# 14 — Frontend Strategy

## Sequence
1. Shared UI kit and design system — Wave A (already delivered).
2. Module screens per Web Solution Designs — waves B → H.

## Shared UI
- App shell, navigation registry, command palette, DataGrid, notifications center, settings pages — all ship in MOD-001.

## Feature Rollout
- Each business module implements Web SD screens sprint-by-sprint.
- Navigation entries are added to the central registry only after RBAC keys land.
