# Milestone Exit Checklist

Applied to every wave before the next wave begins.

- [ ] Architecture review complete (Architecture Board)
- [ ] API contracts finalized (per `docs/60-solution-design/api/`)
- [ ] Database migrations validated (linter clean; rollback tested)
- [ ] Unit tests passing at EEMP Ch. 15 thresholds
- [ ] Integration tests passing
- [ ] Documentation updated (Module Publication + Solution Design deltas)
- [ ] EEMP quality gates satisfied (Ch. 14, 15, 08)
- [ ] Cross-Platform Certification passed
- [ ] Backlog statuses updated in `indexes/master_implementation_backlog.md`
- [ ] Approval recorded (Program Delivery + Architecture Board)

Signed-off checklists are archived under `docs/50-audit-reports/`.
